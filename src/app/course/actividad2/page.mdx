## Taller de conversiones de sistemas numéricos en electrónica digital

### Objetivo:

-   Fortalecer las habilidades en la conversión entre sistemas numéricos binario, decimal y hexadecimal.
-   Aplicar las conversiones de sistemas numéricos a casos prácticos de electrónica digital.

### Guía de ejercicios: (procedimiento por cada ejercicio)

### Parte 1: Conversiones básicas (10 ejercicios)

1.  Convertir el número decimal 728 a binario.
2.  Convertir el número binario 1101 a decimal.
3.  Convertir el número hexadecimal AFE2 a decimal.
4.  Convertir el número decimal 147200 a hexadecimal.
5.  Convertir el número binario 101101001110001 a decimal.
6.  Convertir el número decimal 1502 a hexadecimal.
7.  Convertir el número hexadecimal FA7F a binario.
8.  Convertir el número binario 11110000 a decimal.
9.  Convertir el número decimal 2000 a binario.
10. Convertir el número hexadecimal CAAF a decimal.

### Parte 2: Conversiones aplicadas a electrónica digital (10 ejercicios)

1.  Un código binario de 12 bits se utiliza para representar un valor decimal. ¿Cuál es el rango de valores decimales que se pueden representar?
2.  Un convertidor analógico-digital (ADC) de 10 bits tiene una entrada de voltaje de 5V. ¿Cuál es el valor decimal máximo que puede convertir?
3.  Un convertidor digital-analógico (DAC) de 12 bits tiene una salida de voltaje de 10V. ¿Cuál es el valor decimal que se debe aplicar a la entrada para obtener una salida de 5V?
4.  Un motor tiene una velocidad máxima de 1000 RPM. ¿Cuál es el código binario que se debe aplicar a la entrada de un controlador de motor para que funcione a la cuarta parte de su velocidad máxima con una resolución de 8bit?
5.  Un sensor digital tiene dos estados: 0V para "apagado" y 5V para "encendido". ¿Cuál es el código binario que se debe leer del sensor para saber si está encendido?


### Parte 3: Ejercicios utilizando las unidades de medida de almacenamiento digital:

1. Una memoria USB tiene una capacidad de 8 GB. ¿Cuántas canciones en formato MP3 de 2 MB cada una se pueden almacenar en ella?

2. Un disco duro tiene una capacidad de 500 GB. ¿Cuántas fotos de baja resolución de 1 MB cada una se pueden almacenar en él?

3. Un videojuego tiene un tamaño de 25 GB. ¿Cuánto tiempo se tardaría en descargarlo a una velocidad de internet de 5 Mbps?

4. Un servidor en la nube tiene una capacidad de 50 TB. ¿Cuántos discos duros de 500 GB se necesitarían para almacenar la misma cantidad de información?

5. Una película en definición estándar tiene un tamaño de 4 GB. ¿Cuántas películas se pueden almacenar en un disco duro de 500 GB?

6. Un libro electrónico tiene un tamaño de 500 KB. ¿Cuántos libros electrónicos se pueden almacenar en una memoria USB de 4 GB?

7. Una canción en formato WAV tiene un tamaño de 50 MB. ¿Cuántas canciones se pueden almacenar en un reproductor de música portátil de 2 GB?

8. Un disco duro externo tiene una capacidad de 2 TB. ¿Cuántas horas de música en formato MP3 se pueden almacenar en él?

9. Una cámara digital toma fotos de 5 MP. ¿Cuántas fotos se pueden almacenar en una tarjeta SD de 1 GB?

10. Un software de edición de fotos tiene un tamaño de 500 MB. ¿Cuánto tiempo se tardaría en descargarlo a una velocidad de internet de 2 Mbps?
