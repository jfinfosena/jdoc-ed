## Actividad: Crea tu currículum vitae utilizando solo etiquetas HTML

### Objetivo: Desarrollar un currículum vitae utilizando únicamente etiquetas HTML básicas, sin la ayuda de CSS o JavaScript.

#### Instrucciones:

1.  Estructura básica:
    -   Comienza con la estructura básica de un documento HTML: `<!DOCTYPE html>`, `<html>`, `<head>`, `<title>` (tu nombre), `</head>`, `<body>`, `</body>`, `</html>`.
2.  Encabezado:
    -   Crea una sección de encabezado con tu nombre completo, información de contacto (correo electrónico, teléfono), y un breve resumen profesional.
3.  Experiencia laboral:
    -   Crea una sección para tu experiencia laboral, incluyendo:
        -   Título del puesto: Usa la etiqueta `<h3>`.
        -   Empresa: Usa la etiqueta `<h4>`.
        -   Fecha de inicio y fin: Usa la etiqueta `<p>`.
        -   Descripción de las responsabilidades: Usa una lista sin orden `<ul>` con viñetas para enumerar tus responsabilidades.
4.  Educación:
    -   Crea una sección para tu educación, incluyendo:
        -   Título obtenido: Usa la etiqueta `<h3>`.
        -   Institución educativa: Usa la etiqueta `<h4>`.
        -   Fecha de graduación: Usa la etiqueta `<p>`.
        -   Descripción de cursos relevantes: Usa una lista sin orden `<ul>` con viñetas para enumerar cursos relevantes.
5.  Habilidades:
    -   Crea una sección para tus habilidades, incluyendo:
        -   Habilidades técnicas: Usa una lista sin orden `<ul>` con viñetas para enumerar tus habilidades técnicas.
        -   Habilidades blandas: Usa una lista sin orden `<ul>` con viñetas para enumerar tus habilidades blandas.
6.  Enlaces adicionales:
    -   Si lo deseas, puedes incluir enlaces a tu portafolio online, perfil de LinkedIn u otras páginas web relevantes.
